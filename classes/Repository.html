Class Repository

Repositories will abstract the CRUD operations of your entities. No state is stored in the repository, it only provides methods to fetch, create, update and delete entities. It uses the HttpService to make the requests to the server.

example
import { Repository } from '@euk-labs/fetchx';

const usersRepository = new Repository(httpService, { path: '/users' });

usersRepository.create({ name: 'John Doe', age: 42 });
